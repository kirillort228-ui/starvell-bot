# Starvell Notify Bot

Telegram-бот для уведомлений о новых сообщениях в Starvell.

Бот работает через Next.js JSON-запрос Starvell:

```text
GET https://starvell.com/_next/data/{build_id}/chat.json
```

Он получает список чатов, смотрит `unreadMessageCount`, `user.unreadChatIds` и `lastMessage.id`, а потом присылает уведомление в Telegram с кнопкой «Открыть чат».

## Возможности

- Подключение Starvell аккаунта через Cookie без обязательного прокси.
- Прокси можно добавить позже отдельно для конкретного аккаунта.
- Поддержка прокси:
  - `http://ip:port`
  - `https://ip:port`
  - `socks5://ip:port`
  - `socks5://login:password@ip:port`
- Проверка новых сообщений каждые 60 секунд.
- Ссылка на конкретный чат: `https://starvell.com/chat/{chat_id}`.
- SQLite база данных.
- Включение/выключение уведомлений для каждого аккаунта.

## Установка

```bash
cd starvell_notify_bot
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

На Linux/macOS:

```bash
source .venv/bin/activate
```

## Настройка

Скопируй `.env.example` в `.env`:

```bash
copy .env.example .env
```

На Linux/macOS:

```bash
cp .env.example .env
```

Открой `.env` и вставь токен от BotFather:

```env
BOT_TOKEN=123456:ABCDEF
CHECK_INTERVAL_SECONDS=60
DATABASE_PATH=starvell_bot.db
```

## Запуск

```bash
python bot.py
```

## Как добавить аккаунт

1. Открой бота в Telegram.
2. Нажми `/start`.
3. Нажми `🔐 Мои аккаунты`.
4. Нажми `➕ Добавить аккаунт`.
5. Отправь Cookie от Starvell.

Готово: аккаунт добавится сразу без прокси. Старые сообщения не будут отправлены повторно.

## Как добавить прокси к аккаунту

1. Нажми `🔐 Мои аккаунты`.
2. Под нужным аккаунтом нажми `🌐 Прокси аккаунта`.
3. Отправь прокси в формате `http://ip:port`, `https://ip:port`, `socks5://ip:port` или `socks5://login:password@ip:port`.
4. Чтобы убрать прокси, отправь `-`.

## Где взять Cookie

В Chrome:

1. Открой `starvell.com` и войди в аккаунт.
2. Нажми `F12`.
3. Открой `Network / Сеть`.
4. Нажми `Fetch/XHR`.
5. Открой запрос `chat.json`.
6. В `Headers / Заголовки` найди `Cookie`.
7. Скопируй значение Cookie.

Cookie — это почти как пароль. Не публикуй его, не отправляй в чужие чаты и не выкладывай на GitHub.

## Важные замечания

- При добавлении аккаунта бот сохраняет текущие `lastMessage.id`, чтобы не отправлять старые сообщения.
- Уведомление приходит только при непрочитанном чате.
- Если Starvell изменит структуру сайта или endpoint, клиент может потребовать обновления.
- Cookie может устаревать. Тогда нужно удалить аккаунт в боте и добавить свежий Cookie.

## Пополнение баланса бота

В разделе **👤 Профиль** появилась кнопка пополнения. Сейчас реализовано ручное пополнение:

1. Пользователь нажимает **Пополнить баланс бота**.
2. Пользователь вводит любую сумму в рублях, например `150` или `349.50`. Также доступны быстрые кнопки 100/300/500/1000 ₽.
3. Бот создаёт заявку на пополнение.
4. Пользователь оплачивает по реквизитам из `.env`.
5. Администратор подтверждает заявку кнопкой в Telegram.
6. Баланс пользователя в боте увеличивается.

В `.env` можно указать:

```env
ADMIN_ID=твой_telegram_id
TOP_UP_PAYMENT_DETAILS=перевод на карту 0000 0000 0000 0000, комментарий — ID заявки
```

Если `ADMIN_ID` не указан, бот всё равно создаст заявку, но администратору не придёт кнопка подтверждения.


## Подписка

Если у пользователя нет активной подписки, бот показывает меню:

- 🎁 Попробовать 7 дней бесплатно
- 💰 Пополнить баланс
- 14 дней | 399 ₽
- 1 месяц | 699 ₽ (-15% 🔥)
- 3 месяца | 1399 ₽ (-20% 🔥)
- 6 месяцев | 2699 ₽ (-20% 🔥)

Пробный период можно активировать только один раз. После 7 дней пользователь покупает подписку за баланс бота.


## Безлимитный доступ по Starvell username — устарело

В `.env` можно указать Starvell-юзернеймы, которым бот доступен без подписки:

```env
FREE_USERNAMES=molkyv,Moto_moto214
```

Пользователь должен добавить Starvell-аккаунт в разделе `🔐 Мои аккаунты`.
Теперь используется Telegram username пользователя, а не Starvell username.


## Безлимитный доступ по Telegram username

Бот проверяет именно Telegram username пользователя, а не username Starvell.

В `.env` можно указать Telegram-юзернеймы, которым бот доступен без подписки:

```env
TELEGRAM_FREE_USERNAMES=molkyv,Moto_moto214
```

Писать `@` не обязательно. Пользователь должен иметь username в Telegram.


## Топ продавцов

В главном меню добавлен раздел **🏆 Топ продавцов**.

Что умеет:
- открыть меню топ-продавцов;
- найти профиль продавца по username;
- показать карточку продавца в стиле STARVELL;
- открыть профиль продавца на сайте;
- поделиться ссылкой на профиль.

Для просмотра профиля нужен хотя бы один подключённый Starvell-аккаунт, потому что запросы берутся через Cookie.
